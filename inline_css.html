Generic Font Families

Serif fonts are fonts which include flags on the ends of each letter. Serif fonts are best used when designing reading material such as books and magazine articles.
Sans-Serif fonts do not include the ending flags on each letter and are best when creating data visualizations.
Cursive fonts attempt to mimic handwriting and are best used in activities like signature design.
Fantasy fonts contain a high degree of detail and work well when creating infographics for children.
Monospace fonts occupy an identical amount of space and are commonly preferred by developers when coding.

Color on the Web

RGB (red-green-blue) uses a mixture of 256 color channels for each value to create unique a color.
Hexadecimal colors are represented by a code consisting of letters and numbers or a combination of both to create a six-digit code.
Color Names are a basic form of color identification using text to display color.
